# Chesspoint Custom Move Classification Icons

## Overview

These custom SVG icons are designed specifically for Chesspoint.io to provide unique visual feedback for chess move quality assessment. They feature a modern minimalist design with the Chesspoint color scheme.

## Design Philosophy

These custom SVG icons are designed to be unique and different from standard market icons, featuring:

- **Modern minimalist design** with clean lines and geometric shapes
- **Chesspoint color scheme** using the purple theme colors
- **Consistent visual language** across all classification types
- **High contrast** for excellent readability in both light and dark modes
- **Scalable SVG format** for crisp display at any size

## Technical Features

- **32x32px SVG format** for crisp scaling
- **Consistent stroke weights** (2px borders, 2.5px main elements)
- **Color-coded backgrounds** matching classification severity
- **White foreground elements** for maximum contrast
- **Rounded corners** matching Chesspoint design language
- **Subtle decorative elements** for visual interest

## Usage

These icons are automatically used throughout the Chesspoint application for:

- Move classification displays
- Analysis panels
- Move history
- Game review interfaces

The icons maintain the Chesspoint brand identity while providing clear, intuitive visual feedback for chess move quality assessment.

## File Format

All icons are provided as SVG files for maximum scalability and customization. They can be easily modified or recolored to match different themes or requirements.
